Завдання 1

URL Scheme в JavaScript

У JavaScript схема URL є вбудованим об'єктом, який надає можливість розбити URL-адресу на її компоненти. Давайте розглянемо простий приклад використання схеми URL:

// URL-адреса для аналізу
const urlString = "https://www.example.com/path/to/resource?param1=value1&param2=value2#fragment";


Створення об'єкту URL за допомогою URL-адреси

const url = new URL(urlString);


Отримання різних компонентів URL
 
const protocol = url.protocol; // https:
const hostname = url.hostname; // www.example.com
const pathname = url.pathname; // /path/to/resource
const searchParams = url.searchParams; // { "param1" => "value1", "param2" => "value2" }
const hash = url.hash; // #fragment


Виведення компонентів URL у консоль
 
console.log("Протокол:", protocol);
console.log("Доменне ім'я:", hostname);
console.log("Шлях:", pathname);
console.log("Параметри:", searchParams.toString());
console.log("Фрагмент:", hash);

У цьому прикладі ми створюємо об'єкт URL з рядка urlString, а потім отримуємо доступ до різних компонентів URL, таких як протокол, доменне ім'я, шлях, параметри та фрагмент. Всі ці компоненти можна використовувати окремо для подальшої обробки в JavaScript-коді.
