Завдання №3:

З об'єкту medicines потрібно отримати масив в якому будуть лише назви препаратів. З масиву потрібно прибрати медикаменти , в яких строк зберігання уже пройшов . У новому масиві відсортувати медикаменти у хронологічному порядку. Результат вивести у консоль. Застосувати стрілочні функції const medicines = { Агалгін: new Date("2022-05-01"), Ношпа: new Date("2025-07-02"), Альфахолін: new Date("2024-12-21"), Аспірин: new Date("2022-08-15"), Аспаркам: new Date("2024-04-18"), };

Код JavaScript:

const medicines = { Агалгін: new Date("2022-05-01"), Ношпа: new Date("2025-07-02"), Альфахолін: new Date("2024-12-21"), Аспірин: new Date("2022-08-15"), Аспаркам: new Date("2024-04-18"), }; // Отримуємо масив з назвами препаратів const names = Object.keys(medicines); console.log("Назви препаратів:", names); // Видаляємо медикаменти, строк зберігання яких вже пройшов const validMedicines = Object.entries(medicines) .filter(([name, date]) => date > new Date()) .map(([name, date]) => ({ name, date })); console.log("Дійсні медикаменти:", validMedicines); // Сортуємо медикаменти в хронологічному порядку const sortedMedicines = validMedicines.sort((a, b) => a.date - b.date); console.log("Відсортовані медикаменти:", sortedMedicines);
