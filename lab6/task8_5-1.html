Завдання №8:

Напиши функцію конструктор Storage який створює об'єкти Для управління складом товарів. При виклику отримуватиме один агрумент - початковий масив товарів, і записувати їх у властивість items. Додай методи класу: getItems() - повертайте масив товарів addItems(item) - отримує новий товар та додає його до поточних removeItem(item) - отримує товар і, якщо він є, видаляє його з поточних const arr = ["apple", "banana", "mango"]

Код JavaScript:

function Storage(initialItems) { this.items = initialItems; // Метод для отримання всіх товарів this.getItems = () => { return this.items; }; // Метод для додавання нового товару this.addItem = (item) => { this.items.push(item); }; // Метод для видалення товару this.removeItem = (item) => { const index = this.items.indexOf(item); if (index !== -1) { this.items.splice(index, 1); } }; }

Приклад використання:

// Приклад використання const arr = ["apple", "banana", "mango"]; const storage = new Storage(arr); // Виведення початкового масиву товарів console.log(storage.getItems()); // ["apple", "banana", "mango"] // Додавання нового товару storage.addItem("orange"); console.log(storage.getItems()); // ["apple", "banana", "mango", "orange"] // Видалення товару storage.removeItem("banana"); console.log(storage.getItems()); // ["apple", "mango", "orange"]
