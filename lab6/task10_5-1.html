Завдання №10:

Напишіть функцію checkBrackets(str) яка приймає рядок жс коду (someFn) і перевіряє правильність закриття дужок () {} [] Якщо рядок містить коректний код функція повертає true. В іншому випадку повертає false

Код JavaScript:

function checkBrackets(str) { const brackets = { '(': ')', '[': ']', '{': '}' }; const stack = []; for (let char of str) { if (brackets[char]) { stack.push(brackets[char]); } else if (Object.values(brackets).includes(char)) { if (stack.pop() !== char) { return false; } } } return stack.length === 0; }

Приклад використання:

// Приклади використання функції console.log(checkBrackets("(function(){return [1,2,3,4];})()")); // true console.log(checkBrackets("{[()]}")); // true console.log(checkBrackets("{[(])}")); // false console.log(checkBrackets("{[}")); // false
