Завдання №2:

Напишіть функцію getProductDetails, яка приймає ідентифікатор товару productId та дві колбек-функції successCallback та errorCallback. // Функція getProductDetails повинна отримати деталі про вказаний товар та передати їх у successCallback. // У випадку якщо товар не знайдено, викликається errorCallback і передається повідомлення про помилку.

Код JavaScript:

// Приклад бази даних товарів const productsDatabase = { 1: { id: 1, name: "Laptop", price: 1000 }, 2: { id: 2, name: "Phone", price: 500 }, 3: { id: 3, name: "Tablet", price: 300 } }; // Функція getProductDetails function getProductDetails(productId, successCallback, errorCallback) { // Симуляція асинхронного запиту до бази даних setTimeout(() => { const product = productsDatabase[productId]; if (product) { successCallback(product); } else { errorCallback(`Product with ID ${productId} not found.`); } }, 1000); // Симуляція затримки в 1 секунду } // Колбек-функція успіху function onSuccess(product) { console.log("Product details:", product); } // Колбек-функція помилки function onError(errorMessage) { console.error("Error:", errorMessage); }

Приклад використання:

// Перевірка функції getProductDetails getProductDetails(1, onSuccess, onError); // Має вивести деталі товару з ID 1 getProductDetails(4, onSuccess, onError); // Має вивести помилку, оскільки товару з ID 4 немає
